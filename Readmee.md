Project description:
-This project showcases various button styles and animations using CSS. The buttons are designed with different hover and active states to enhance user interaction and provide visual feedback.

Design choices:  
-Ensured consistent styling and behavior across different button classes for uniformity.
-Incorporated visual feedback on hover and active states to improve user experience.
-Kept the code clean and simple for easy understanding and modification.
-Changed button shapes to have rounded corners to make them more appealing to the users.

Challenges faced:
-Balancing padding changes and background color transitions to ensure smooth animations.

Features to be Proud Of: 
-The buttons provide clear visual feedback on user actions, improving the overall interactivity.
-The CSS code is organized and easy to understand, making it simple to add new button styles or modify existing ones.
